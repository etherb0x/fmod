## [model.ADSRModulator](#modeladsrmodulator)

An AHDSR curve [model.Modulator](scripting-api-reference-project-model-modulator.html#modelmodulator) applied to a single property of a [model.AutomatableObject](scripting-api-reference-project-model-automatableobject.html#modelautomatableobject).

**See Also:** [AHDSR Modulator](https://fmod.com/docs/2.02/studio/modulator-reference.html#ahdsr-modulator)

## [model.AutopitchModulator](#modelautopitchmodulator)

A [model.Modulator](scripting-api-reference-project-model-modulator.html#modelmodulator) that affects the pitch of an instrument across the range of [model.GameParameter](scripting-api-reference-project-model-parameter.html#modelgameparameter)s.

**See Also:** [Autopitch Modulator](https://fmod.com/docs/2.02/studio/modulator-reference.html#autopitch-modulator)

## [model.LFOModulator](#modellfomodulator)

A low frequency oscillator [model.Modulator](scripting-api-reference-project-model-modulator.html#modelmodulator) that applies a function to a single property of a [model.AutomatableObject](scripting-api-reference-project-model-automatableobject.html#modelautomatableobject).

**See Also:** [LFO Modulator](https://fmod.com/docs/2.02/studio/modulator-reference.html#lfo-modulator)

## [model.Modulator](#modelmodulator)

The base class for objects which additively affect the value of a [model.AutomatableObject](scripting-api-reference-project-model-automatableobject.html#modelautomatableobject)'s property.

**See Also:** [Modulator](https://fmod.com/docs/2.02/studio/modulator-reference.html)

## [model.RandomizerModulator](#modelrandomizermodulator)

A [model.Modulator](scripting-api-reference-project-model-modulator.html#modelmodulator) that applies random values to a single property of a [model.AutomatableObject](scripting-api-reference-project-model-automatableobject.html#modelautomatableobject).

**See Also:** [Random Modulator](https://fmod.com/docs/2.02/studio/modulator-reference.html#random-modulator)

## [model.SidechainModulator](#modelsidechainmodulator)

A [model.Modulator](scripting-api-reference-project-model-modulator.html#modelmodulator) that uses the input [signal](glossary.html#signal) from an external [model.Sidechain](scripting-api-reference-project-model-mixereffect.html#modelsidechain) to apply random values to a single property of a [model.AutomatableObject](scripting-api-reference-project-model-automatableobject.html#modelautomatableobject).

**See Also:** [Sidechain Modulator](https://fmod.com/docs/2.02/studio/modulator-reference.html#sidechain-modulator), [Sidechain Effect](https://fmod.com/docs/2.02/studio/glossary.html#sidechain-effect)
