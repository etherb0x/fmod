# 26\. Scripting API Reference

This chapter provides a reference for [FMOD Studio](glossary.html#fmod-studio)'s scripting API, and assumes some basic knowledge of programming concepts and JavaScript. For more information about running scripts, and for example scripts, see the [Scripting](scripting-terminal-reference.html) chapter.
